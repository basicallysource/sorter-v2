## Bill of Materials

This machine is currently in development, and thee parts list may change at any time. The most up-to-date <a href="https://docs.google.com/spreadsheets/d/1aoxcyK0xa0i3iWYlO4u3-vsruOGU8V_U9XT65YhMZmY/edit?gid=0#gid=0">Bill of Materials</a> is here. A summary is below:
- 

Aluminum Extrusions.
The structure of the machine is supported by T Slot 2020 Aluminum Extrusion profile, which you can buy from a variety of sources such as <a href="https://www.amazon.com/Aluminum-Extrusion-Rail-Easy-Workbenches-Furniture/dp/B09XM7RT6V/">Amazon</a>.

Table of lengths:
Feeder system:
- 3x 270mm: Bulk bucket supports
- 3x 190mm: Mid C-Channel supports
- 3x 110mm: Lower C-Channel supports
- Either/or:
  - Fixed camera support: - 1x 300mm
  - Adjustable camera support: 1x 350mm + 1x 200mm

Interface layer:
- 244mm: top beam supports

Each bin layer:
- 6x 158: radial supports
- 6x 320mm: edge crossbeams

Vertical supports:
- 6x 280mm + (160mm * number of bin layers): the interface layer is 280mm, and each bin layer adds 160mm
